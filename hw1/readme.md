### Зад.1
####Да се напише функция (GCD a b), която намира най-големия общ делител на две естествени числа по метода на Евклид:

```
function gcd(a, b)
    while a ≠ b
        if a > b
           a := a − b
        else
           b := b − a
    return a
```
